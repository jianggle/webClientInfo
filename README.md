# webClientInfo

A method that get web client info.

It return json like this:

```
{
  "browser": {
    "name": "Chrome",
    "ver": "83.0.4103.116"
  },
  "os": {
    "name": "Windows",
    "ver": "10"
  },
  "ua": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/83.0.4103.116 Safari/537.36"
}
```

## Install

```
# Using npm
npm i web-client-info -S
```
